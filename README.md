## ✨ Overview

**Gaze** is more than just a Pomodoro timer. It’s a native Windows utility that utilizes the top edge of your display to create a dynamic drop-down focus HUD. 

Featuring a procedural **Mascot** that tracks your cursor with its eyes, Gaze adds a layer of interactive life to your screen while keeping you on track with beautiful Mica materials and smooth, system-native animations.

## 🌟 Key Features

- 📱 **Dynamic Screen-Edge Integration**: A persistent, non-intrusive HUD that hugs the top of your monitor.
- 👁️ **The Gaze Mascot**: A smooth, 2D procedural animation that watches your every move (cursor tracking).
- 🌊 **Fluid Animations**: Built with modern UI physics for that fluid native feel.
- 🛠️ **Sidebar Control Center**: A clean, modern settings interface utilizing Windows 11 design principles.
- ⚡ **Cinematic Intro**: Experience a stunning neon-grid welcome every time you launch the app.
- 🔄 **Auto Updates**: Secure, automatic software updates to keep your experience fresh.

## 🚀 Getting Started

### Installation
1. Download the latest **`Gaze_Installer.exe`** from the [Releases](https://github.com/AtharvaBari/Gaze-Windows/releases) page.
2. Run the installer and follow the prompt instructions.
3. Open the app and grant any necessary permissions (required for the mascot to track your cursor globally).

### For Developers
Gaze is built using modern C# standards and .NET 8.

```bash
# 1. Clone the repo
git clone https://github.com/AtharvaBari/Gaze-Windows.git
cd Gaze-Windows

# 2. Open the solution
# Double-click Gaze.sln to open in Visual Studio 2022

# 3. Build and Run
# Set the main project as the startup project and press F5
```

## 🛠️ Tech Stack

  - **UI Framework:** WinUI 3 (Windows App SDK) / WPF
  - **Animation:** WinUI Composition APIs
  - **Persistence:** Entity Framework Core / SQLite
  - **Lifecycle:** C# async/await & Task Parallel Library
  - **Distribution:** MSIX / Squirrel
